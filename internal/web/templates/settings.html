Anonymous API Keys

Simple bearer tokens that grant access to the proxy. Requests using these keys are authenticated but tracked as unknown in the stats dashboard — no per-key breakdown is available. Use these for quick setups or tools where you don't need per-app visibility. Stored under server.api_keys in the config file.
